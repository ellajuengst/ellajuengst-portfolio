React and Firebase Trivia Web App


summary

In my web development course, I was tasked with creating a web app using an API and backend to store user data. My group project partner and I decided to use a public trivia API which can be found here. We pull trivia questions from the API for users to practice their trivia skills. Users can choose from different categories, levels of difficulty, and types of questions. They can save questions that they enjoy to their account page and edit their user information. We implemented the front end using React framework and Bootstrap for basic styling. I then further customized the styles by adding my own custom CSS. For the back end, we used Firebase to store all our user information and saved questions.
